@page install How to Install

Gly engine tries to have a simple installation with as little effort as possible, just downloading a single file command interface with the entire framework built in,
with examples and assets.

## NodeJS

```
wget https://cdn.jsdelivr.net/npm/@gamely/gly-cli@latest/bin/cli.js
```

```
node cli.js help
```

## Lua 

```
wget http://get.gamely.com.br/cli.lua
```

```
lua cli.lua help
```
